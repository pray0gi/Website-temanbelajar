Implementasi di Dunia Nyata (Python)

Di Python, struktur data `list` adalah sebuah **Array Dinamis**. Ia menggabungkan kehebatan keduanya: akses data yang cepat seperti Array, dan ukuran yang fleksibel seperti List.

# 1. Membuat list (kosong atau langsung diisi)
daftar_belanja = ["Gula", "Kopi"]

# 2. Menambah data (fleksibel seperti List)
daftar_belanja.append("Teh")         # Menambah di akhir
daftar_belanja.insert(1, "Sabun")  # Menyisipkan di indeks 1

# 3. Mengakses data (cepat seperti Array)
print(daftar_belanja[0]) # Output: Gula

# 4. Menghapus data
daftar_belanja.remove("Kopi")

# Hasil akhir
print(daftar_belanja)
# Output: ['Gula', 'Sabun', 'Teh']
