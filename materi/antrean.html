Konsep Dasar Antrean

Antrean atau Queue adalah struktur data linear yang mengikuti prinsip **First-In, First-Out (FIFO)**. Artinya, elemen yang pertama kali ditambahkan ke dalam antrean akan menjadi elemen yang pertama kali dikeluarkan. Bayangkan ini persis seperti antrean orang di loket tiket: orang yang datang lebih dulu akan dilayani lebih dulu.

Operasi Utama: Enqueue

Menambahkan elemen baru ke bagian paling belakang (akhir) dari antrean.

Operasi Utama: Dequeue

Menghapus dan mengembalikan elemen dari bagian paling depan antrean.

Simulasi: Antrean Tiket Bioskop

LOKET

Contoh Penggunaan di Dunia Nyata

  • Antrean Print Dokumen: Saat Anda mencetak beberapa dokumen, printer akan memprosesnya sesuai urutan pengiriman.
  • Penjadwalan Tugas (Task Scheduling): Sistem operasi menggunakan antrean untuk menentukan proses CPU mana yang akan dieksekusi selanjutnya.
  • Layanan Pelanggan: Sistem chat atau telepon layanan pelanggan menangani permintaan sesuai urutan masuknya.
  • Buffering Data: Dalam streaming video, data video disimpan sementara dalam antrean sebelum ditampilkan untuk memastikan pemutaran yang lancar.

Contoh Implementasi

JavaScript (ES6 Class)


class Queue {
  constructor() {
    this.items = [];
  }

  // Menambah elemen
  enqueue(element) {
    this.items.push(element);
  }

  // Menghapus elemen
  dequeue() {
    if (this.isEmpty()) {
      return "Antrean kosong";
    }
    return this.items.shift();
  }

  // Memeriksa antrean
  isEmpty() {
    return this.items.length === 0;
  }
}

// Contoh penggunaan:
const antreanLoket = new Queue();
antreanLoket.enqueue("Adi");
antreanLoket.enqueue("Budi");
console.log(antreanLoket.items); 
// Output: ["Adi", "Budi"]

antreanLoket.dequeue();
console.log(antreanLoket.items); 
// Output: ["Budi"]

Python (collections.deque)


from collections import deque

# Cara paling efisien untuk
# membuat Queue di Python
antrean_loket = deque()

# Menambah elemen (enqueue)
antrean_loket.append("Adi")
antrean_loket.append("Budi")

print(antrean_loket)
# Output: deque(['Adi', 'Budi'])

# Menghapus elemen (dequeue)
# elemen pertama akan dihapus
antrean_loket.popleft()

print(antrean_loket)
# Output: deque(['Budi'])

# Memeriksa antrean
if not antrean_loket:
  print("Antrean kosong")
